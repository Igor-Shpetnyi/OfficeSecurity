# ADR-0002: Гібридна LLM-модель

**Рішення:** хмарний API (GPT-4o-mini / Claude Haiku) як основний класифікатор, fallback на чисті RegEx-правила при недоступності API (circuit breaker, п.13 ТЗ).

**Чому:** точність класифікації складних векторів ("транзит повз" vs "курс на Суми") у локальних 7-8B моделей помітно нижча, різниця в ціні — кілька доларів на місяць. Тримати GPU-сервер заради економії $5/міс — погана угода часу розробника.

**Умова перегляду:** якщо бюджет стає категорично 0 — переходити на локальну LLM, закладаючи +2-3 тижні на інтеграцію/тюнінг.

**Ризик і мінімізація:** rate limit LLM API під час масованої атаки → fallback на RegEx (сирий сигнал все одно йде, з міткою "⚠️ без ШІ-верифікації").

**Статус:** accepted.
